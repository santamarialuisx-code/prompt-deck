# Anime Avatar from Selfie

## Prompt

Transform this selfie into a high-quality anime avatar. Maintain the subject's key facial features (hair color, eye shape, distinguishing marks) while applying clean anime art style with cel-shading. Use vibrant, saturated colors with soft gradient shading. The expression should be natural and flattering. Add a subtle glow effect around the character for a polished, professional look suitable for Discord and gaming platforms.

## Tips

- Front-facing selfies with clear facial features work best
- Specify hair color and style if the AI doesn't detect it accurately
- Add "masterpiece, best quality" to the prompt for higher fidelity

## Negative Prompt

low quality, deformed face, extra fingers, blurry, pixelated
