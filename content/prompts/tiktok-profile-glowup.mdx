# TikTok Profile Glow-Up

## Prompt

Transform this selfie into a trendy TikTok profile photo. Apply bold, saturated color grading with a slight teal-orange split tone. Add a soft glow effect and subtle lens flare for a cinematic feel. Enhance the eyes slightly for more impact. The overall vibe should be energetic, youthful, and scroll-stopping. Crop for vertical (9:16) profile display.

## Tips

- Selfies with a bit of attitude or expression work best
- The teal-orange color grade is trending and universally flattering
- Works great for creators, influencers, and casual users alike

## Negative Prompt

boring, corporate, stiff, low energy, washed out
